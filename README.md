### Quizz tool for french conjugation practice
This project contains both a nodejs backend and a React frontend
first install and run the backend:

```bash
cd backend
npm install
npm start
```
The backend will run on port 5000 by default.

Then, in another terminal, install and run the frontend:

```bash
cd frontend
npm install
npm start
```
The frontend will run on port 3000 by default and will proxy requests to the backend.
