### familiarize with the backend
read the following sources, and their code comments
```bash
/README.md
/server/app.js
/server/routes/conjugations.js
/server/index.js
/server/package.json
/server/tests/app.test.js
```
try to run
```bash
    cd server
    npm install
    npm run test
```
keep the backend running
```bash
    npm start
```
open a new terminal for the frontend
### familiarize with the frontend
read the following sources, and their code comments
```bash
/client/package.json
/client/public/index.html
/client/src/index.js
/client/src/App.js
```
try to run
```bash
    cd client
    npm install
    npm run
```
and click on the "click Me" button to see an example of value fetched from the backend
